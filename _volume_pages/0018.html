{% raw %}XIV{% endraw %}
{% raw %}LANGUAGE.{% endraw %}
{% raw %}The traveller entering Italy for the first time should do so, not{% endraw %}
{% raw %}by rail, but by one of the Alpine passes (Spliigen, Simplon, etc.),{% endraw %}
{% raw %}as only thus will he obtain an adequate idea of the full ethno¬{% endraw %}
{% raw %}graphical significance of the Alps, which conceal so new and so{% endraw %}
{% raw %}strange a world from northern Europe. The luxurious character of{% endraw %}
{% raw %}the Italian climate, vegetation, and scenery, the soft richness of{% endraw %}
{% raw %}the language, and the courtly manners of the upper classes all{% endraw %}
{% raw %}present a striking contrast to the harsher and rougher characteristics{% endraw %}
{% raw %}of German Switzerland or the Tyrol. On no account, however.{% endraw %}
{% raw %}should he traverse these passes at night, and he should always in¬{% endraw %}
{% raw %}form himself beforehand of the condition of the diligence, and{% endraw %}
{% raw %}raise an energetic protest against broken windows and similar in¬{% endraw %}
{% raw %}conveniences. In spring it is advisable to wear coloured spectacles{% endraw %}
{% raw %}as a precaution against the dazzling reflection from the extensive{% endraw %}
{% raw %}snow-fields (p. xxvi).{% endraw %}
{% raw %}The traveller who has entered Italy by one of the Alpine passes{% endraw %}
{% raw %}is recommended to quit the country via Nice (1 day), Cannes{% endraw %}
{% raw %}O/2 day)> Marseilles (1 day), Aries (Y2 day), Ntmes (1 day), Avignon{% endraw %}
{% raw %}(1 day), and Lyons (R. 1), all of which are worthy of a visit, even{% endraw %}
{% raw %}after Italy.{% endraw %}
{% raw %}III. Language.{% endraw %}
{% raw %}The time and labour which the traveller has bestowed on the{% endraw %}
{% raw %}study of Italian at home will be amply repaid as he proceeds on his{% endraw %}
{% raw %}journey. Is is quite possible for persons entirely ignorant of Italian{% endraw %}
{% raw %}and French to travel through Italy with tolerable comfort; but such{% endraw %}
{% raw %}travellers cannot conveniently deviate from the ordinary track, and{% endraw %}
{% raw %}are moreover invariably made to pay lalla Inglese' by hotel-keepers{% endraw %}
{% raw %}and others, i. e. considerably more than the ordinary charges. French{% endraw %}
{% raw %}is very useful, as the Italians are very partial to that language, and{% endraw %}
{% raw %}it may suffice for Rome and some of the main routes; but for those{% endraw %}
{% raw %}who desire the utmost possible freedom, and who dislike being im¬{% endraw %}
{% raw %}posed upon, a slight acquaintance with the language of the country{% endraw %}
{% raw %}is indispensable, f{% endraw %}
{% raw %}t 'Baedekers Manual of Conversation in English, French, German, and{% endraw %}
{% raw %}Italian, -with Vocabulary, etc.'' (Stereotype Edition), which is specially{% endraw %}
{% raw %}adapted for the use of travellers, with the addition of a pocket-diction¬{% endraw %}
{% raw %}ary, will soon enable the beginner to make himself understood. —{% endraw %}
{% raw %}A few words on the pronunciation may be acceptable to persons unac¬{% endraw %}
{% raw %}quainted with the language. C before e and i is pronounced like the{% endraw %}
{% raw %}English ch; g before e and i like j. Before other vowels c and g are{% endraw %}
{% raw %}hard. Ch and gh, which generally precede e or t, are hard. Sc before e{% endraw %}
{% raw %}or i is pronounced like sh; gn and gl between vowels like nyi and lyi.{% endraw %}
{% raw %}The vowels a, e, i, 0, u are pronounced ah, a, ee, o, 00. — In ad¬{% endraw %}
{% raw %}dressing persons of the educated classes 'Ella' or 'Lei1, with the 3rd pers.{% endraw %}
{% raw %}King., should always be employed (addressing several at once, 'loro1 with{% endraw %}
{% raw %}the 3rd pers. pi). 'Voi1 is used in addressing waiters, drivers, etc., Hu'{% endraw %}
{% raw %}by those only who are proficient in the language. 'Voi1 is the usual mode{% endraw %}
{% raw %}of address among the Neapolitans, but is generally regarded as inelegant{% endraw %}
{% raw %}or uneourteous.{% endraw %}
