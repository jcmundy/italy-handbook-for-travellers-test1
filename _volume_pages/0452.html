{% raw %}S. Stefano. BOLOGNA. 47. Route. 315{% endraw %}
{% raw %}On the left in the Via S. Stefano is situated —{% endraw %}
{% raw %}*Sto. Stefano (PL F, 5), a pile consisting of seven different{% endraw %}
{% raw %}churches, containing ancient columns and mural paintings, and,{% endraw %}
{% raw %}according to an inscription still extant (of which there is a copy{% endraw %}
{% raw %}on the exterior to the left, of 1769), occupying the site of a temple{% endraw %}
{% raw %}of Isis, having probably been founded in the 5th century. The{% endraw %}
{% raw %}churches are not all on the same level, the last having been con¬{% endraw %}
{% raw %}structed as a crypt below the first. The first church, of 1637, contains{% endraw %}
{% raw %}nothing noteworthy. — A chapel leads thence to the left into the{% endraw %}
{% raw %}second church (now undergoing alteration), *S. Sepolcro, originally{% endraw %}
{% raw %}the Baptistery, surrounded by an ambulatory, and erected for its{% endraw %}
{% raw %}present purpose before the year 1000. In the 12th cent, the tomb{% endraw %}
{% raw %}of St. Petronius was added in imitation of the Holy Sepulchre at{% endraw %}
{% raw %}Jerusalem, and a brick column was placed adjacent to each of the{% endraw %}
{% raw %}antique marble columns. The ambo with the symbols of the evan¬{% endraw %}
{% raw %}gelists is of the 9th century. Over the altar are frescoes of the 15th{% endraw %}
{% raw %}cent, by an unknown master. — To the left is situated the third{% endraw %}
{% raw %}church (also in course of alteration), that of SS. Pietro e Paolo,{% endraw %}
{% raw %}an ancient basilica, in which the last column to the right of the{% endraw %}
{% raw %}altar has an antique Ionic capital. The altar adjoining the choir{% endraw %}
{% raw %}on the left is a sarcophagus dating from the 9th cent., and adorned{% endraw %}
{% raw %}with a cross between two peacocks; it contains the bones of the{% endraw %}
{% raw %}martyr St. Vitalis (d. 382). Above it, a Madonna with saints by{% endraw %}
{% raw %}Lor. Sabbatino (d. 1577). On the right the sarcophagus of the{% endraw %}
{% raw %}martyr Agricola (9th cent.), the saint being represented with wings,{% endraw %}
{% raw %}between a stag and a lion; above it a crucifix by Simone da Bo¬{% endraw %}
{% raw %}logna, surnamed de' Crocifissi (14th cent.). — We now return to{% endraw %}
{% raw %}the second church, whence the first door on the left leads us to{% endraw %}
{% raw %}the fourth church, the Atrio di Pilato; in the centre is a font with{% endraw %}
{% raw %}an inscription mentioning the Lombard king Luitprand (d. 744).{% endraw %}
{% raw %}Chapel on the left, Crucifixion by Franc. Francia. To the right of{% endraw %}
{% raw %}the portal is the fifth church (below the first), della Confessione,{% endraw %}
{% raw %}dating from the 11th century. — The sixth church, della Conso-{% endraw %}
{% raw %}lazione, adjoins the Atrio di Pilato on the right. — Lastly, on the{% endraw %}
{% raw %}left, rises the seventh church, della Trinith, resting on pillars, in{% endraw %}
{% raw %}the centre of which is a series of columns with Romanesque capitals.{% endraw %}
{% raw %}3rd Chapel on the right, an Adoration of the Magi in wood (14th{% endraw %}
{% raw %}cent.). — To the right the handsome cloisters (13th cent.) of the{% endraw %}
{% raw %}suppressed Celestine monastery.{% endraw %}
{% raw %}The Piazza in front of the church contains several handsome{% endraw %}
{% raw %}Renaissance buildings. A side-street to the right now leads to —{% endraw %}
{% raw %}S. Giovanni in Monte (PL F, 5, 6), one of the oldest churches{% endraw %}
{% raw %}in Bologna, founded by St. Petronius in 433, re-erected in the{% endraw %}
{% raw %}Gothic style in 1440 , and restored in 1824. It consists of a low{% endraw %}
{% raw %}nave with aisles and a short transept, and a facade and dome of{% endraw %}
{% raw %}more recent date.{% endraw %}
