{% raw %}PREFACE.{% endraw %}
{% raw %}The objects of the Handbook for Italy, which consists of{% endraw %}
{% raw %}three volumes, each complete in itself, are to supply the trav¬{% endraw %}
{% raw %}eller with some information regarding the culture and art of{% endraw %}
{% raw %}the people he is about to visit, as well as regarding the nat¬{% endraw %}
{% raw %}ural features of the country, to render him as independent as{% endraw %}
{% raw %}possible of the services of guides and valets-de-place, to pro¬{% endraw %}
{% raw %}tect him against extortion, and in every way to aid him in{% endraw %}
{% raw %}deriving enjoyment and instruction from his tour in one of{% endraw %}
{% raw %}the most fascinating countries in the world. The Handbook{% endraw %}
{% raw %}will also, it is hoped, be the means of saving the traveller{% endraw %}
{% raw %}many a trial of temper; for there is probably no country in{% endraw %}
{% raw %}Europe where the patience is more severely taxed than in{% endraw %}
{% raw %}some parts of Italy.{% endraw %}
{% raw %}The Handbook is founded on the Editor's personal ac¬{% endraw %}
{% raw %}quaintance with the places described, most of which he has{% endraw %}
{% raw %}repeatedly and carefully explored. As, however, changes{% endraw %}
{% raw %}are constantly taking place, he will highly appreciate any{% endraw %}
{% raw %}communications with which travellers may kindly favour{% endraw %}
{% raw %}him, if the result of their own observation. The information{% endraw %}
{% raw %}already received from numerous correspondents, which he{% endraw %}
{% raw %}gratefully acknowledges, has in many cases proved most{% endraw %}
{% raw %}serviceable.{% endraw %}
{% raw %}The present volume, corresponding to the ninth German{% endraw %}
{% raw %}edition, has, like its predecessor, been thoroughly revised and{% endraw %}
{% raw %}considerably augmented. Its contents have been divided into{% endraw %}
{% raw %}groups of routes arranged historically and geographically{% endraw %}
{% raw %}(Piedmont, Liguria, Lombardy, Venetia, The Emilia, and{% endraw %}
{% raw %}Tuscany), each group being provided with a prefatory outline{% endraw %}
{% raw %}of the history of the district. Each section is also prefaced{% endraw %}
{% raw %}with a list of the routes it contains, and may be removed{% endraw %}
{% raw %}from the volume and used separately if desired.{% endraw %}
