{% raw %}From the S.E. corner of the Piazza of St. Mark to the Lagune,{% endraw %}
{% raw %}extends the *Piazzetta (PI. F, 4), which is bounded on the W. by{% endraw %}

{% raw %}The *Lihrary (Antica Libreria di S. Marco), which now belongs{% endraw %}
{% raw %}to the royal palace (see p. 237), begun by Sansovino in 1536, is a{% endraw %}
{% raw %}magnificent structure of the 16th cent., and one of the finest secular{% endraw %}
{% raw %}edifices in Italy. In plan the structure consists of a double colonnade{% endraw %}
{% raw %}with arches and embedded columns. In the upper colonnade the{% endraw %}
{% raw %}arches rest upon smaller, additional columns of the Ionic order. The{% endraw %}
{% raw %}effect is so fine, that Sansovino may be fairly said to have justified{% endraw %}
{% raw %}the liberty he has taken in enlarging the metopes at the expense of{% endraw %}
{% raw %}the triglyphs and architrave, and in some other points (Burckhardt).{% endraw %}

{% raw %}by the Doge Michiel from Syria in 1120, and erected here in 1180;{% endraw %}

{% raw %}the tutelary saint of Venice; the other is surmounted by St. TheoÂ¬{% endraw %}

{% raw %}in 1329. This used to be the place of execution, and is now the{% endraw %}

{% raw %}between the Library and the Royal Garden, is situated the Zecca or{% endraw %}
{% raw %}Mint, which was also built by Sansovino in 1536.{% endraw %}
