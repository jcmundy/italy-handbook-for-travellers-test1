{% raw %}The Casa di Ricceri (PI. H, 4), in the Via della Pergola, which{% endraw %}
{% raw %}skirts the E. side of the Spedale S. Maria Nuova, was once occu¬{% endraw %}
{% raw %}pied by Benvenuto Cellini.{% endraw %}
{% raw %}From the Via della Pergola the Via degli Alfani leads to the{% endraw %}
{% raw %}N.W. to the church of S. Maria degli Angeli (PL G, 4), the cloisters{% endraw %}
{% raw %}of which contain frescoes by Andrea del Castagno, and to the Palazzo{% endraw %}
{% raw %}Giugni, built by Ammanati in 1560, with a fine court.{% endraw %}
{% raw %}To the S.E. of the Via della Pergola, and parallel to it, runs{% endraw %}
{% raw %}the Via Borgo Pinti, in which is situated the church of S. Maria{% endraw %}
{% raw %}Maddalena de' Pazzi (PL H, 5). The cloisters in front of the church{% endraw %}
{% raw %}were designed by Giuliano da Sangallo; the columns were modelled{% endraw %}
{% raw %}on an ancient capital found at Fiesole (1479). In the 2nd chapel,{% endraw %}
{% raw %}on the left, is a Coronation of the Madonna by Cosimo Rosselli; in{% endraw %}
{% raw %}the 4th chapel, two saints, by Raffaellino del Garbo. — In the ad¬{% endraw %}
{% raw %}jacent Via della Colonna, No. 1, is the entrance to the chapter¬{% endraw %}
{% raw %}house of the monastery belonging to the church, which contains a{% endraw %}
{% raw %}large *Fresco by Perugino (Christ on the Cross , date about 1500,{% endraw %}
{% raw %}the most worthy representation of the Saviour by this master), and{% endraw %}
{% raw %}is open daily, 12-4 (25 c, on Sun. free). — In the same street,{% endraw %}
{% raw %}No. 31, is the exhibition of the Societa d'Incoraggiamento delle{% endraw %}
{% raw %}Belle Arti (open daily, 10-4; admissin 60 c). — In the Palazzo{% endraw %}
{% raw %}della Crocetta (PL H, 4), Via della Colonna 26, a large Archaeo¬{% endraw %}
{% raw %}logical Museum is now being fitted up ; the antique sculptures are{% endraw %}
{% raw %}on the ground-floor, the Egyptian and Etruscan antiquities and the{% endraw %}
{% raw %}collection of coins and gems on the first floor (adm., see p. 369).{% endraw %}
{% raw %}In the Via Borgo Pinti, No. 62, farther N., is the Palazzo Pan¬{% endraw %}
{% raw %}ciatichi- Ximenes, erected by Giuliano da Sangallo in 1490, and{% endraw %}
{% raw %}restored in the 17th cent, by Silvani. It contains a collection of{% endraw %}
{% raw %}Japanese porcelain, of weapons, and of pictures, including Ma¬{% endraw %}
{% raw %}donnas by Fra Fiiippo Lippi and Fra Diamante (for sale).{% endraw %}
{% raw %}The Via della Colonna connects the Piazza dell' Annunziata{% endraw %}
{% raw %}(p. 410) and the new Piazza d'Azeglio (PL H, I, 5), which is laid{% endraw %}
{% raw %}out in promenades. — To the S. of the latter are a handsome new{% endraw %}
{% raw %}Synagogue, and, farther on, the church of S. Ambrogio (PL H, 6).{% endraw %}
{% raw %}On the right, 1st chapel, Cosimo Rosselli, Madonna with saints and{% endraw %}
{% raw %}angels; 2nd and 3rd chapels, pictures of the school of Spinello Aretino.{% endraw %}
{% raw %}To the left in the choir, a 'Canopy by Mino da Fiesole (1482), and a{% endraw %}
{% raw %}large fresco by Cosimo Rosselli.{% endraw %}
{% raw %}The Galleria Buonarroti, see p. 409; S. Croce, see p. 406.{% endraw %}

{% raw %}de' Gondi to the right, which leads us to the Piazza S. Firenze{% endraw %}
{% raw %}(PI. F, 5), with the church of that name. No 1 in this Piazza is{% endraw %}
{% raw %}the Palazzo Gondi, begun in 1490 by Giuliano da Sangallo, and{% endraw %}
{% raw %}completed in 1874 by Poggi, with a rustica 1'ac.ade and a handsome{% endraw %}
{% raw %}court. In one of the rooms on the first floor is a marble chimney-{% endraw %}
{% raw %}piece, with a relief by G. da Sangallo. From this point the Via del{% endraw %}
{% raw %}Phoconsolo (PI. F, 5) diverges to the Piazza del Duomo.{% endraw %}
