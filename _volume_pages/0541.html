{% raw %}392 Route 53. FLORENCE. Biblioteca Nazionale.{% endraw %}
{% raw %}The first floor of the edifice contains the Biblioteca Nazionale{% endraw %}
{% raw %}(admission, see p. 369; entrance by the 8th door from the piazza),{% endraw %}
{% raw %}which has been formed since 1860 by the union of the grand-ducal{% endraw %}
{% raw %}Biblioteca Palatina formerly in the Pitti Palace, and the still more{% endraw %}
{% raw %}extensive Biblioteca Magliabecchiana. The latter, founded by An¬{% endraw %}
{% raw %}tonio Magliabecchi, a jeweller of Florence, has been dedicated to{% endraw %}
{% raw %}the use of the public since 1747. The present library contains{% endraw %}
{% raw %}about 300,000 vols, and 8000 MSS., comprising the most important{% endraw %}
{% raw %}works of the literature of other nations. There are also several{% endraw %}
{% raw %}very rare impressions: the first printed Homer, Florence 1488;{% endraw %}
{% raw %}Cicero ad Familiares, Venice 1469; Dante, by Landino (Florence{% endraw %}
{% raw %}1481, in a handsome binding adorned with niellos); Piero Medici's{% endraw %}
{% raw %}presentation copy of the Anthologia Graeca (Ed. Lascaris). Every{% endraw %}
{% raw %}facility is afforded for the use of the library; to the right, at the{% endraw %}
{% raw %}end of the great reading-saloon, is the room containing the cata¬{% endraw %}
{% raw %}logues. — The staircase to the right of the library leads to the{% endraw %}
{% raw %}^Central Archives of Tuscany (PI. D, 5), arranged by Bonaini,{% endraw %}
{% raw %}one of the most imposing collections of this description (140,000{% endraw %}
{% raw %}documents).{% endraw %}
{% raw %}Between the Uffizi and the Palazzo Vecchio is the Via della{% endraw %}
{% raw %}Ninna, leading to the E. to the Via del Leone, in which is situated{% endraw %}
{% raw %}the Loggia del Grano (PI. E, 6), erected by Giulio Parigi in 1619{% endraw %}
{% raw %}and adorned with a bust of Cosimo II.{% endraw %}
{% raw %}From the post-office the Via Lambertesca leads to the Via Por{% endraw %}
{% raw %}S. Maria, which ends at the Ponte Vecchio. At the corner to the{% endraw %}
{% raw %}left rises the Torre dei Girolami, near which is the old church of{% endraw %}
{% raw %}Sto. Stefano (PL E, 5, 6), where Boccaccio, by desire of the Signoria{% endraw %}
{% raw %}publicly explained Dante's Divine Comedy in 1373.{% endraw %}
{% raw %}The Borgo SS. Apostoli leads hence to the W. to the small{% endraw %}
{% raw %}Piazza del Limbo , in which stands the church of SS. Apostoli{% endraw %}
{% raw %}(PI. D, 5), an early-Tuscan basilica of the 11th cent., with an in¬{% endraw %}
{% raw %}scription (on the facade, to the left) attributing its foundation to{% endraw %}
{% raw %}Charlemagne. The arches between the nave and aisles are adorned{% endraw %}
{% raw %}with a fine ancient border and rest upon columns with well-exe¬{% endraw %}
{% raw %}cuted composite capitals. The aisles are vaulted. At the end of{% endraw %}
{% raw %}the left aisle is a ciborium by Andrea della Robbia, adjoining which{% endraw %}
{% raw %}is the monument of Oddo Altoviti, by Benedetto da Rovezzano.{% endraw %}
{% raw %}The sculptured decoration of the portal is also by the latter artist.{% endraw %}
{% raw %}The right aisle contains the tomb of Bindo Altoviti, by Ammanati{% endraw %}
{% raw %}(1570). ■— From this point to S. Trinita and Via Tornabuoni,{% endraw %}
{% raw %}see p. 428.{% endraw %}
{% raw %}The Por S. Maria is also connected with the Via Tornabuoni by{% endraw %}
{% raw %}the Via delle Terme, at the beginning of which, on the right stands{% endraw %}
{% raw %}the old Almshouse of the Capitani della Parte Guelfa.{% endraw %}
{% raw %}The Piazza S. Firenze, the Badia, and the Museo Nazionale{% endraw %}
{% raw %}see pp. 400-405; by the Ponte Vecchio to the Pal. Pitti, see p. 429.'{% endraw %}
