{% raw %}54 Route 8. LAIBACH. From Vienna{% endraw %}
{% raw %}224 M. Cilli (787 ft. ; *Erzherzog Johann; Kaiserkrone; Lowe),{% endraw %}
{% raw %}an ancient town of some importance, founded by Emp. Claudius{% endraw %}
{% raw %}(Claudia Celleia), contains several Roman reliefs and memorial slabs{% endraw %}
{% raw %}on the town-walls. On a wooded height in the vicinity stands the{% endraw %}
{% raw %}ruined castle of Obercilli.{% endraw %}
{% raw %}The train crosses the green Sann, and enters the narrow and{% endraw %}
{% raw %}wooded valley of that stream. The most picturesque part of the{% endraw %}
{% raw %}whole line is between Cilli and Sava. 22972 M. Markt Tuffer,{% endraw %}
{% raw %}with a ruined castle. 234 M. Rbmerbad (which memorial stones{% endraw %}
{% raw %}prove to have been known to the Romans), also called Teplitza(i. e.{% endraw %}
{% raw %}'warm bath'), a beautifully situated watering-place.{% endraw %}
{% raw %}240 M. Steinbruck (*Rail. Restaurant; 25 min. allowed for{% endraw %}
{% raw %}express passengers to dine in going to Vienna), a thriving village{% endraw %}
{% raw %}on the Save, or Sou, which here unites with the Sann, is the{% endraw %}
{% raw %}junction for the line to Agram and Karlstadt. The train now runs{% endraw %}
{% raw %}for 1 hr. in the narrow valley of the Save, enclosed by lofty lime¬{% endraw %}
{% raw %}stone cliffs, which often barely afford space for the river and rail¬{% endraw %}
{% raw %}way. 245 M. Hrastnigg ; 247 M. Trifail, with valuable coal-mines;{% endraw %}
{% raw %}250 M. Sagor, the first place in Carniola; 254 M. Sava.{% endraw %}
{% raw %}The valley now expands. At Littai the Save is crossed. Scen¬{% endraw %}
{% raw %}ery still very picturesque. Stations Kressnitz, Loose. At the{% endraw %}
{% raw %}influx of the Laibach into the Save , the line quits the latter and{% endraw %}
{% raw %}enters the valley of the former. The lofty mountain-range to the{% endraw %}
{% raw %}N.W. is that of the Julian or Carnian Alps. 274 M. Salloch.{% endraw %}
{% raw %}278 M. Laibach (940 ft.; *Stadt Wien; *Elephant; Europa; *Rail.{% endraw %}
{% raw %}Restaurant), Slav. Ljubljana, on the Laibach, the capital of Car¬{% endraw %}
{% raw %}niola, with 26,300 inhab. , is situated in an extensive plain en¬{% endraw %}
{% raw %}closed by mountains of various heights. An old Castle, now used{% endraw %}
{% raw %}as a prison, rises above the town. The Cathedral, in the Italian{% endraw %}
{% raw %}style, is decorated with stucco and frescoes of the 18th century.{% endraw %}
{% raw %}The line now traverses the marshy Laibacher Moos by means{% endraw %}
{% raw %}of an embankment, 13/4 M. in length, and crosses the Laibach,{% endraw %}
{% raw %}which becomes navigable here, although hardly 3 M. below the{% endraw %}
{% raw %}point where it issues from the rocks near Oberlaibach. Near{% endraw %}
{% raw %}(292 M.) Franzdorf the line crosses a lofty viaduct, and enters a{% endraw %}
{% raw %}more mountainous district. — 302 M. Loitsch (1555 ft.; Post or Stadt{% endraw %}
{% raw %}Triest). About 15 M; to the N.W. of Loitsch are the rich quicksilver{% endraw %}
{% raw %}mines of Idria. — 308 M. Ralcek, 3^/2 M. to the S.E. of which is{% endraw %}
{% raw %}the Zirknitzer See, enclosed by lofty mountains. 318 M. Adelsberg{% endraw %}
{% raw %}(1798 ft. ; Adelsberger Hof; Krone), Slav. Post6jno.{% endraw %}

{% raw %}accidentally re-discovered in 1816, are 3/4 M. to the W. of Adelsberg The{% endraw %}
{% raw %}fees are fixed by tariff, and are somewhat high for a single visitor (from{% endraw %}

{% raw %}party. Brilliant illumination is necessary in order to produce a satisfactorv{% endraw %}
{% raw %}effect. A visit to the grotto occupies 272-3 hrs., or if prolonged to the{% endraw %}
