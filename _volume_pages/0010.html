{% raw %}VI{% endraw %}
{% raw %}PREFACE.{% endraw %}
{% raw %}The Editor is indebted to Professor A. Springer for the in¬{% endraw %}
{% raw %}troductory article on Art, which has special reference to Nor¬{% endraw %}
{% raw %}thern Italy and Florence, and for the art-historical notices pre¬{% endraw %}
{% raw %}fixed to the description of the larger towns and principal pic¬{% endraw %}
{% raw %}ture-galleries. The admirable works of Messrs. Crowe and Ca-{% endraw %}
{% raw %}valcaselle have also been laid extensively under contribution.{% endraw %}
{% raw %}The Maps and Plans , upon which special care has been{% endraw %}
{% raw %}bestowed, will abundantly suffice for the use of the ordinary{% endraw %}
{% raw %}traveller.{% endraw %}
{% raw %}Heights are given in English feet (1 Engl. ft. = 0,3048{% endraw %}
{% raw %}metre), and Distances in English miles (comp. p. ii). The{% endraw %}
{% raw %}Populations are given from the most recent official sources.{% endraw %}
{% raw %}Hotels (comp. p. xviii). Besides the modern palatial and{% endraw %}
{% raw %}expensive establishments, the Handbook also contains a se¬{% endraw %}
{% raw %}lection of modest, old-fashioned inns, which not unfrequently{% endraw %}
{% raw %}afford good accommodation at moderate charges. The asterisks{% endraw %}
{% raw %}indicate those hotels which the Editor has reason to believe{% endraw %}
{% raw %}from his own experience, as well as from information supplied{% endraw %}
{% raw %}by numerous travellers, to be respectable, clean, and reason¬{% endraw %}
{% raw %}able. The value of these asterisks, it need hardly be observed,{% endraw %}
{% raw %}varies according to circumstances, those prefixed to town{% endraw %}
{% raw %}hotels and village inns signifying respectively that the{% endraw %}
{% raw %}establishments are good of their kind. At the same time the{% endraw %}
{% raw %}Editor does not doubt that comfortable quarters may occasion¬{% endraw %}
{% raw %}ally be obtained at inns which he has not recommended or{% endraw %}
{% raw %}even mentioned. The average charges are stated in accord¬{% endraw %}
{% raw %}ance with the Editor's own experience, or from the bills{% endraw %}
{% raw %}furnished to him by travellers. Although changes frequently{% endraw %}
{% raw %}take place, and prices generally have an upward tendency,{% endraw %}
{% raw %}the approximate statement of these items which is thus{% endraw %}
{% raw %}supplied will at least enable the traveller to form an estimate{% endraw %}
{% raw %}of his probable expenditure.{% endraw %}
{% raw %}To hotel-proprietors, tradesmen, and others the Editor{% endraw %}
{% raw %}begs to intimate that a character for fair dealing and courtesy{% endraw %}
{% raw %}towards travellers forms the sole passport to his commend¬{% endraw %}
{% raw %}ation, and that advertisements of every kind are strictly ex¬{% endraw %}
{% raw %}cluded from his Handbooks.{% endraw %}
