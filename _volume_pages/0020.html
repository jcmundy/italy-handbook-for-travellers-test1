{% raw %}Many shops now profess to have fixed prices, but even in these{% endraw %}
{% raw %}cases it is usual to offer two-thirds or three-quarters only of the{% endraw %}
{% raw %}price demanded. The same rule applies to artizans, drivers, and{% endraw %}
{% raw %}others. lNon volete?' (then you will not?) is a remark which gener¬{% endraw %}
{% raw %}ally has the effect of bringing the matter to a speedy adjustment.{% endraw %}
{% raw %}Purchases should never be made by the traveller when accompanied{% endraw %}
{% raw %}by a valet-de-place. These individuals, by tacit agreement, receive{% endraw %}
{% raw %}from the seller at least 10 per cent of the purchase-money, a bonus{% endraw %}
{% raw %}which of course comes out of the pocket of the purchaser.{% endraw %}
{% raw %}The traveller should always be abundantly supplied with cop¬{% endraw %}
{% raw %}per coin in a country where trifling donations are in constant{% endraw %}
{% raw %}demand. Drivers, guides, and other persons of the same class in¬{% endraw %}
{% raw %}variably expect, and often demand as their right, a gratuity (buona{% endraw %}
{% raw %}mano, mancia, da bere, bottiglia, caffe, fumata) in addition to the{% endraw %}
{% raw %}hire agreed on, varying according to circumstances from 2-3 sous{% endraw %}
{% raw %}to a franc or more. The traveller need have no scruple in limiting{% endraw %}
{% raw %}his donations to the smallest possible sums, as liberality frequently{% endraw %}
{% raw %}becomes a source of annoyance and embarrassment. Thus, if half-{% endraw %}
{% raw %}a-franc is bestowed where two sous would have sufficed, the fact{% endraw %}
{% raw %}speedily becomes known , and the donor is sure to be besieged by{% endraw %}
{% raw %}numerous other applicants whose demands it is impossible to satisfy.{% endraw %}
{% raw %}In Northern Italy the traveller will now find comparatively few{% endraw %}
{% raw %}causes for complaint, as the system of fixed charges is gradually{% endraw %}
{% raw %}being introduced at the hotels and the shops. He will generally{% endraw %}
{% raw %}find the people with whom he comes in contact civil and obliging,{% endraw %}
{% raw %}and if he has some acquaintance with the language he will rarely{% endraw %}
{% raw %}meet with attempts at extortion.{% endraw %}
{% raw %}VII. Railways.{% endraw %}
{% raw %}Northern Italy is now overspread with so complete a network of{% endraw %}
{% raw %}railways that the traveller will seldom use any other conveyance,{% endraw %}
{% raw %}except on the Alpine routes and on the lakes. The rate of travel¬{% endraw %}
{% raw %}ling is very moderate, and the trains are often behind time. The{% endraw %}
{% raw %}first class carriages are tolerably comfortable, the second are inferior{% endraw %}
{% raw %}to those of the German railways, and resemble the English and{% endraw %}
{% raw %}French, while the third class is chiefly frequented by the lower{% endraw %}
{% raw %}orders. Among the expressions with which the railway-traveller will{% endraw %}
{% raw %}soon become familiar are — lprontV (ready), '■partenza' (departure),{% endraw %}
{% raw %}lsi cambia convoglio1 (change carriages), and luscitd' (egress).{% endraw %}
